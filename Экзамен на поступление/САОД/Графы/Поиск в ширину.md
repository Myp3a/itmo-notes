1. Создаем список посещенных узлов
2. Создаем список известных узлов
3. В список известных помещаем начальный узел
4. Для всех известных узлов
	1. Добавить в список посещенных
	2. Убрать из списка известных
	3. Для каждого соседа
		1. Если не посещен и неизвестен - добавить в список известных
		2. Если нужный нам - завершить
5. Перейти к шагу 4
